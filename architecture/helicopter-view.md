## 2. Классификация поддоменов (ось «конкурентного преимущества»)

```text
   высокое  │  ┏━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┓
конкурентное│  ┃           C O R E   (это и есть Gonka)        ┃
преимущество│  ┃  • Proof of Compute 2.0  (x/inference)        ┃
            │  ┃  • Off-chain оркестрация (dapi)               ┃
            │  ┃  • Devshard (растущее ядро монетизации)       ┃
            │  ┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛
            │  ┌───────────────────────────────────────────────┐
            │  │           S U P P O R T I N G                  │
            │  │  collateral · bls · streamvesting             │
            │  │  restrictions · genesistransfer               │
            │  └───────────────────────────────────────────────┘
            │  ┌ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ┐
            │    G E N E R I C   (берётся готовым / форк)
   низкое   │  │ cosmos-sdk: staking* group gov bank feegrant  │
            │     bookkeeper · CometBFT      (*перепрошит)
            └  └ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ┘
              └──────────── строй сам ───────── покупай готовое ──────────►
```

> Главный приём: `x/staking` **перепрошит** так, что `voting power = PoC score`
> (а не токены). Generic-модуль превращён в носитель Core-идеи.

## 4. Главные часы: один такт — много исполнителей (EndBlock)

```text
   t ──►  StartOfPoC ─► Generation ─► Validation ─► EndOfPoCValidation ─► SetNewValidators ─► ClaimMoney
              │             │             │                    │                    │                 │
              │             │             │      ┌─────────────┴─────────────┐      │                 │
           seed +       MMR-коммит    SPRT-аудит │ ComputeNewWeights         │  цена unit         recovery
           группы       (off-chain)   честности  │ settlement · slash        │  SetComputeVal      → награды
                                                 │ BLS DKG · делегирование   │  (активация H+2)
                                                 └───────────────────────────┘
   ── сбой СОСЕДА → epoch_error (цепь живёт);  сбой ЯДРА эпохи → return err (halt) ──
```
